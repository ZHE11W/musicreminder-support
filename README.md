# 隐私政策

最后更新：2025年9月

## 信息收集
醒记 App 不会收集、存储或传输您的任何个人信息。

## 数据存储
- 所有提醒数据仅存储在您的设备本地
- 录音文件仅保存在您的设备上
- 我们无法访问您的任何数据

## 权限使用
- **麦克风权限**：仅用于录制提醒语音，录音文件不会离开您的设备
- **通知权限**：仅用于在设定时间发送提醒通知
- **音频播放**：仅用于播放您设置的提醒音乐

## 第三方服务
醒记 App 不使用任何第三方分析或广告服务。

## 数据安全
您的所有数据都安全地存储在您的设备上，我们无法访问。
